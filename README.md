## Getting started

### Requirements

This projects depends on:

* [CMake](http://www.cmake.org/install/)
* OpenGL
* GLEW (version `2.0.0 +`)
* GLFW (version `3.2.1 +`)
* GLM (version `0.9.8.3 +`)


#### Mac OS X 

Install the dependencies using [Homebrew](http://brew.sh/). Run these 
commands:

```bash
brew install cmake
brew install glew
brew install glfw3
brew install glm
```
